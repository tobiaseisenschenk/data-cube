question_answer How it works

UX Data-Cube is a community effort to make small UX-Data big. The main goal of this project is to feed a database for UX-Studies to allow for more data-driven decisions on which UX-Method Mixes to use when. Step 1: Set up an account by simply using your email Step 2: Provide anonymous data about your current software project Step 3: Come back once you did any sort of Usability Testing or UX-Study Step 4: Share anonymous meta-data about your study Step 5: Profit from access to all existing UX-Data

whatshot Features

add Access to a collective database of UX-Studies add Facilitates documentation of important variables from the project environment add Captures Data from Usability Studies add Managing your own (data) contributions add Graphical Represenation of the data add Export of a filtered data subset
